# customer-churn-retention-sql
sql project to analyze customer churn and retention using real business scenarios
## customer churn & retention analytics system

### business problem 
customer churn leads to revenue loss . This project uses sql to identify early churn signals so businesses can retain customers proactively.

### project objective 
- detect customer at risk of churn
- analyze retention pattern
- provide actionable insights for business decisions


### key features 
- retention and cohort analystics
- customer lifetime value (clv)
- plan-wise churn insights


### tools used
- sql
- github
